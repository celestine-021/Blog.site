# Screenshots Directory

This directory should contain screenshots of the blog application:

- `homepage.png` - Screenshot of the main blog page
- `components.png` - Diagram showing component relationships
- `mobile-view.png` - Mobile responsive view
- `article-detail.png` - Individual article view (if implemented)

To add screenshots:
1. Take screenshots of your running application at http://localhost:5173
2. Save them in this directory with the appropriate names
3. The README.md references these images
